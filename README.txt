(Sorry for this inconvenient because I use PC, but I have put chromedriver for running on MAC, dont worry!)

1. Set MAVEN_HOME, JAVA_HOME on your machine
2. Clone automation repo to your machine.
3. Install Chrome ver99 on your machine
4. Open Cmd/or Termial and navigate to project folder, input "mvn clean install test -Dsuite=newjob" and press Enter to see how it works.
5. Open "src/test/java/reports/" to see the report (includes screenshot if failed)
